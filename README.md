## **安裝**

可以使用 Composer 來安裝 model-orderbywith。請在你的專案目錄中運行以下命令：

```
composer require jerry58321/model-orderbywith
```

## **說明**

這是程式庫是 Laravel 中 Eloquent Builder 的擴充方法，
它可以實現透過 Eloquent Relationships 的方式關聯排序。

除此之外：
1. 可以使用聚合函數排序
2. 可以使用遠程關聯排序
3. 可以在關聯中加上其他條件
